### 🎯 **အသုံးပြုနည်းအဆင့်များ**

#### **1️⃣ ရောဂါများ Browse ခြင်း**
```
1. "📚 Browse" tab သို့ သွားပါ
2. ရောဂါအုပ်စုရွေးချယ်ပါ
   ├─ ဗိုင်းရပ်စ်ရောဂါ
   ├─ အဖြူရောင်ဝင်ခြင်း
   ├─ အိုင်းအေ့စ်ကြောင့်ရောဂါ
   └─ အည်အုံခြင်း
3. ရောဂါကိုရွေးချယ်ပါ
4. အသေးစိတ်အချက်အလက်ကြည့်ပါ
```

#### **2️⃣ ရောဂါရှာစေတင်ခြင်း**
```
1. အစဥ်လိုင်းတွင် ရောဂါအမည်ထည့်သွင်ပါ
2. အလိုက်အလမ်းရေကြည်းတွေကြည့်ပါ
3. ရောဂါရွေးချယ်ပါ
```

#### **3️⃣ လက်ခဏာစူးစမ်းခြင်း**
```
1. "🔍 Checker" tab သို့ သွားပါ
2. သင်ခံစားနေသောလက်ခဏာများကိုရွေးချယ်ပါ
   ├─ ခေါင်းကိုက်ခြင်း
   ├─ အအေးခြင်း
   ├─ ချစ်စုံ
   ├─ အည်ခြင်း
   └─ အည်အာခြင်း
3. "🔍 စူးစမ်းခြင်း" ခလုံးနှိပ်ပါ
4. ကျေးဇူးပြုသည့်ရောဂါများကြည့်ပါ
```

#### **4️⃣ ရောဂါယှဉ်ပြိုင်ခြင်း**
```
1. "📊 ယှဉ်ပြိုင်ခြင်း" tab သို့ သွားပါ
2. အခြင်းအရေး၊ ကူးစက်ခြင်း တွဲယူကြည့်ပါ
3. အုံမုံအချိန်နှင့် ကြာချိန်ယှဉ်ပြိုင်ခြင်း
```

#### **5️⃣ အတားအဆစ်နည်းများ**
```
1. "🛡️ အတားအဆစ်" tab သို့ သွားပါ
2. ရောဂါအုပ်စုအလိုက်တားဆစ်နည်းများကြည့်ပါ
3. အုံခြင်းအတားအဆစ်နည်းများလုပ်ဆောင်ပါ
```
